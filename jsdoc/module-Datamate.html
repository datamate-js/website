(static) makeArea(_x, _y, _width, _height, _yArea)

Initialize DatamateArea following with given parameters. DatamateArea object makes rectangle areas the number of given xy blocks. We can bind a DatamateArea to each column with bindArea() method. We can extract a DatamateArea that is binded to a column with area() method. DatamateArea has a information of positions related to the arranged rectangle area such as x, y, width, height, left, right, top, bottom, centerX and centerY. When we use them as the origin of related position, it is easy to reuse the same positioning program just change some variables..
Parameters:
Name Type Default Description
_x Number The left position of whole area rectangle.
_y Number The top position of whole area rectangle.
_width Number The width of the size of whole area rectangle.
_height Number The height of the size of whole area rectangle. // * @param {Number} _xArea The number of blocks to be divided horizontally.
_yArea Number 1 The number of blocks to be divided vertically.

(static) setValues(_param, _values)

Make a new column and values set, or load dataset from the file in given file path. When we set the column name to _param and set the values array to _values. The make new or reset values in the column. When we set a file path to _param and set null to _values, start to load and parse the file as the dataset immediately. In column make mode and if given columnName does not exist, a new column will be created and set given values in it. If given columnName exists, the exsist values will be overwritten as given new values.
Parameters:
Name Type Default Description
_param String | Number column name to be set, or related file path to be loaded..
_values Array null values as an Array to be set.
